# Lissage-figures-with-Matplotlib
<b>A Python-based visualization of Lissajous curves that procedurally generates new frequency ratios in real-time using Matplotlib.</b> <br>
<i>If you want 2D, open main.py. If you want to see 3D visualisation, run main3d.py.</i>
